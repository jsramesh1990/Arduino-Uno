## 🔧 Hardware Assembly Instructions

### Step-by-Step Assembly Guide

#### **Step 1: Prepare the Workspace**
```
✓ Workbench with ESD protection
✓ Soldering station (if using PCB)
✓ Multimeter
✓ Oscilloscope (optional but recommended)
✓ Wire strippers and cutters
✓ Screwdrivers set
✓ Anti-static wrist strap
✓ Component organizer boxes
```

#### **Step 2: Power Supply Circuit Assembly**
```
1. Mount LM2596 voltage regulator on PCB
2. Add input filtering capacitor (100µF/25V)
3. Connect 12V input connector
4. Add output capacitor (220µF/10V)
5. Set output voltage to 5V using trim pot
6. Verify output with multimeter

Component Values:
- C1: 100µF/25V (Input filtering)
- C2: 220µF/10V (Output filtering)
- C3: 0.1µF (Decoupling)
- D1: 1N5819 (Protection diode)
- R1, R2: Voltage divider (1k/2.2k for 5V)
```

#### **Step 3: Arduino Uno Mounting**
```
1. Place Arduino Uno on PCB with standoffs
2. Connect VIN to 12V input
3. Connect GND to common ground
4. Add decoupling capacitor near VIN
5. Verify power LED illuminates
6. Test programming via USB
```

#### **Step 4: Sensor Module Installation**

**A. LiDAR Sensor (VL53L0X)**
```
1. Mount VL53L0X on PCB
2. Connect SDA to A4, SCL to A5
3. Connect VCC to 3.3V
4. Add pull-up resistors (4.7kΩ) on SDA/SCL
5. Verify I2C communication
6. Test with simple sketch
```

**B. GPS Module (NEO-6M)**
```
1. Mount NEO-6M on PCB
2. Connect TX→D4, RX→D3
3. Connect VCC to 5V
4. Connect active antenna
5. Ensure clear view of sky for testing
6. Test using serial monitor
```

**C. Fingerprint Sensor (R307)**
```
1. Mount R307 with flex cable
2. Connect TX→D5, RX→D6
3. Connect VCC to 5V
4. Test with enrollment sketch
5. Calibrate sensor sensitivity
```

**D. RFID/NFC Reader (MFRC522)**
```
1. Mount MFRC522 on PCB
2. Connect SPI pins (D10-D13)
3. Connect RST to D9
4. Connect VCC to 3.3V
5. Test with read card sketch
6. Place antenna at edge of PCB
```

#### **Step 5: Communication Modules**

**A. CAN Bus Module (MCP2515)**
```
1. Mount MCP2515 on PCB
2. Connect SPI pins (D10-D13)
3. Connect CAN_H and CAN_L
4. Add 120Ω termination resistor
5. Test with loopback mode
6. Verify with CAN analyzer
```

**B. LoRa Module (SX1278)**
```
1. Mount SX1278 module
2. Connect SPI pins
3. Connect RST to D8, DIO0 to D2
4. Connect 868MHz antenna
5. Test with ping-pong sketch
6. Check RSSI and SNR values
```

**C. GSM Module (SIM800L)**
```
1. Mount SIM800L with care
2. Connect TX→D7, RX→D8
3. Connect VCC to 5V (with capacitor)
4. Insert SIM card
5. Connect GSM antenna
6. Test AT commands
```

#### **Step 6: Test Points and Indicators**
```
1. Add status LEDs:
   - Red: Power OK
   - Green: System Ready
   - Blue: Communication Active
   - Yellow: Error/Warning

2. Add test points:
   - TP1: 5V rail
   - TP2: 3.3V rail
   - TP3: GND reference
   - TP4: Analog input
   - TP5: PWM output

3. Add push buttons:
   - Reset button
   - Test start button
   - Calibration button
   - User programmable button
```

## 📊 Hardware Testing Procedure

### Pre-Power Test Checklist

```
□ Check all power connections
□ Verify polarity of all components
□ Check for shorts using multimeter
□ Confirm voltage regulator output
□ Ensure all ICs are oriented correctly
□ Check solder joints for cold solder
□ Verify connections with continuity test
□ Remove all power before modifications
```

### Power-On Test Sequence

```cpp
// Hardware Test Sequence
void hardwareTestSequence() {
    // 1. Power Supply Test
    Serial.println("Testing Power Supply...");
    float v5 = readVoltage(5V_PIN);
    float v3 = readVoltage(3V3_PIN);
    
    if (abs(v5 - 5.0) < 0.1 && abs(v3 - 3.3) < 0.1) {
        Serial.println("✓ Power Supply OK");
        LED_GREEN_ON();
    } else {
        Serial.println("✗ Power Supply Error");
        LED_RED_BLINK();
    }
    
    // 2. I2C Bus Test
    Serial.println("Testing I2C Bus...");
    if (scanI2C() > 0) {
        Serial.println("✓ I2C Bus OK");
        LED_BLUE_ON();
    } else {
        Serial.println("✗ I2C Bus Error");
        LED_RED_BLINK();
    }
    
    // 3. SPI Bus Test
    Serial.println("Testing SPI Bus...");
    if (testSPI()) {
        Serial.println("✓ SPI Bus OK");
        LED_BLUE_ON();
    } else {
        Serial.println("✗ SPI Bus Error");
        LED_RED_BLINK();
    }
    
    // 4. UART Test
    Serial.println("Testing UART Ports...");
    if (testUART()) {
        Serial.println("✓ UART Ports OK");
        LED_GREEN_ON();
    } else {
        Serial.println("✗ UART Error");
        LED_RED_BLINK();
    }
}
```
